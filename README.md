## Getting Started
### Prerequisites

* node 18.0.0
* yarn
* install yarn if don't have yet
  ```sh
  npm install --global yarn
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/nguyenvuong1309/hotel_booking_client.git
   ```
2. Install yarn packages
   ```sh
   yarn install
   ```
3. run 
   ```sh
   yarn dev:dev
   ```
